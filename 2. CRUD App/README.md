# CRUD App

Connect to the *Chinook.sqlite* database in this folder using the connection string: `DataSource=Chinook.sqlite;Cache=Shared;`.

Create a simple ASP.NET Razor Pages application using .NET version 10.0 with individual authentication in a folder named *Chinook.Web* that works with the chinook database anmed *Chinook.sqlite*. I want:

1. the data that currently exists in the *Chinook.sqlite* should not be lost
2. build all the crud pages for the Genre table
3. add a link to the Genre list on the main menu
