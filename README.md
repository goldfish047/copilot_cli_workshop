# Coding with Copilot CLI

**Learn how to use GitHub Copilot CLI to boost your development workflow — all from your terminal, using plain English.**

In this workshop, you will follow a complete mini development lifecycle, building a working web app connected to a real database, and documenting it entirely using AI.

**`0. Setup` → `1. Connect` → `2. Build` → `3. Document`**

---

## [0. Setup](./0.%20Setup/README.md)

Get your environment ready. Install everything you need to follow along with the tutorials.

**You will need:**
- A GitHub account
- .NET 10
- SQLite
- GitHub Copilot CLI
- PowerShell 7+ *(Windows only)*

---

## [1. Connect to a Database](./1.%20Connect%20Database/README.md)

Use plain English to connect to the Chinook database and explore its tables and data (no SQL knowledge required).

**What you will do:**
- Connect to the Chinook SQLite database
- List and explore its tables
- Ask Copilot for insights about the data

---

## [2. Build a CRUD App](./2.%20CRUD%20App/README.md)

Build a working ASP.NET Razor Pages web app on top of the Chinook database, step by step, one prompt at a time.

**What you will do:**
- Create an ASP.NET Razor Pages project
- Connect it to the Chinook database using Entity Framework
- Scaffold full CRUD pages for the Genre table
- Apply a custom Bootswatch Sketchy theme

---

## [3. Documentation](./3.%20Documentation/README.md)

Generate real project documentation, audit your app for security risks, and plan what version 2 could look like.

**What you will do:**
- Insert fake sample data into the database
- Generate a README for the project
- Run a security audit saved to SECURITY.md
- Plan the next version of your app

---
